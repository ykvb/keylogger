# Keylogger
A simple Python script that logs keyboard and mouse activity on Windows systems.
## Features
- Logs all key presses withtimestamps
- Tracks active window and process
- Records mouse clicks and coordinates
- Saves logs to temp directory
- Runs in background silently
## Warning
Use responsibly! - Only monitor systems you own or have permission to monitor.
## Installation
```
# Clone this repo
git clone https://github.com/ykvb/keylogger.git
cd input-monitor

# Install dependencies
pip install -r requirements.txt
```
## Usage
```
# Run the monitor
python main.py

# Press Ctrl+C to stop
```
